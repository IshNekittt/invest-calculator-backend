# 📈 API інвестиціонного калькулятору (Монте Карло & GBM)

RESTful API для фінансового калькулятора, що використовує метод Монте-Карло на базі Геометричного броунівського руху (GBM) для стохастичного прогнозування капіталу.

### 🛠 Стек технологій

- **Node.js & Express.js** — маршрутизація та обробка запитів.
- **MongoDB & Mongoose** — збереження історії розрахунків.
- **Математика** — Лема Іто, Перетворення Бокса-Мюллера, `Float64Array` для оптимізації 2000 симуляцій.

### 📍 Маршрути (Endpoints)

#### 1. Розрахунок моделі

`POST /api/calculate`
**Payload (JSON):** Вимагає `deviceId` та `inputs` (initial, monthly, expectedReturn, volatility, inflation, years).
Вбудована cувора перевірка вхідних типів та обмежень для захисту від переповнення Event Loop.

```json
{
  "deviceId": "uuid-string",
  "inputs": {
    "initial": 10000,
    "monthly": 500,
    "expectedReturn": 10,
    "volatility": 15,
    "inflation": 3,
    "years": 20
  }
}
```

**Response (201 Created):** Повертає об'єкт зі згенерованим `chartData` (масив точок графіка за роками) та `summary` (фінальний результат 10/50/90 процентилів).

#### 2. Отримання історії

`GET /api/history/:deviceId`
**Response (200 OK):** Повертає масив останніх 10 розрахунків користувача за його `deviceId` (за винятком великих даних діаграм), відсортованих за датою.
