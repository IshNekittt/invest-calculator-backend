# 📈 API инвестиционного калькулятора (Монте Карло & GBM)

RESTful API для финансового калькулятора, использующего метод Монте-Карло на базе Геометрического броуновского движения (GBM) для стохастического прогнозирования капитала.

### 🛠 Стек технологий

- **Node.js & Express.js** — маршрутизация и обработка запросов.
- **MongoDB & Mongoose** — сохранение истории расчетов.
- **Математика** — Лемма Ито, Преобразование Бокса-Мюллера, `Float64Array` для оптимизации 2000 симуляций.

### 📍 Маршруты (Endpoints)

#### 1. Расчет модели

`POST /api/calculate`
**Payload (JSON):** Требуется `deviceId` и `inputs` (initial, monthly, expectedReturn, volatility, inflation, years).
Встроена строгая валидация входящих типов и лимитов для защиты от переполнения Event Loop.

```json
{
  "deviceId": "uuid-string",
  "inputs": {
    "initial": 10000,
    "monthly": 500,
    "expectedReturn": 10,
    "volatility": 15,
    "inflation": 3,
    "years": 20
  }
}
```

**Response (201 Created):** Возвращает `chartData` для рендеринга коридора вероятности и метрик `summary`.

#### 2. Получение истории

`GET /api/history/:deviceId`
**Response (200 OK):** Возвращает последние 10 расчетов пользователя по его `deviceId` (за исключением больших данных диаграмм), отсортированных по дате.
